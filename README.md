# WATONOMOUS ASD ONBOARDING TASK

## Completed by: Rudra Shukla, Aabjosh Singh, Bhavya Kurseja

### What it does:
Avoids obstacles in a foxglove environment using a virtual LiDAR sensor, costmap amd A* algorithm for object avoidance. Implemented in ROS2.
